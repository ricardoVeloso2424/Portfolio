# Ricardo Veloso Portfolio

Personal portfolio built with Next.js to present my background, technical skills, and selected software projects.

## Stack

- Next.js (App Router)
- React + TypeScript
- Tailwind CSS
- Framer Motion

## Run Locally

```bash
npm install
npm run dev
```

The app runs by default at `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## Technical Highlights

This portfolio is intentionally built to demonstrate:

- A maintainable Next.js App Router structure with clear separation of page composition, UI components, and static portfolio data
- Responsive UI implementation with reusable section and project card components
- Production-oriented frontend quality standards (TypeScript, ESLint, image optimization, accessible focus states)
- Lightweight interaction design with reduced-motion support and performance-conscious animation choices
